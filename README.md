# موقعي الشخصي

موقع شخصي مبني باستخدام React و TypeScript، مع لوحة تحكم لإدارة المحتوى.

## المميزات

- تصميم عصري وجذاب
- لوحة تحكم كاملة لإدارة المحتوى
- دعم كامل للغة العربية
- تأثيرات حركية جميلة
- واجهة مستخدم سهلة الاستخدام
- تجربة مستخدم سلسة

## التقنيات المستخدمة

- React
- TypeScript
- Vite
- Mantine UI
- Framer Motion
- Express.js (Backend)

## التشغيل المحلي

1. تثبيت الاعتمادات:
```bash
npm install
```

2. تشغيل الخادم:
```bash
cd server
npm install
npm run dev
```

3. تشغيل واجهة المستخدم:
```bash
npm run dev
```
